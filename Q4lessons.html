LESSON 1 CODES
Alert box
• Usually used when you want to make sure that the user
reads your message such as a warning or a greeting.
• Syntax: alert (“message”)
User Input Prompt
• This is usually used when you want data to be prompted
first before the user can enter a page.
• The user will have to choose either “ok” or “cancel”
to continue. If the user chooses “ok”, the prompt box
will return the encoded value; else, it will return null.
ò Syntax: prompt(“message”,”defaultvalue”)
Confirm box
• This is usually used when you want the user to
verify or accept something.
• The user will have to choose either “ok” or “cancel”.
If “ok” was chosen, the confirm box will return true;
else, it will return false.
• Syntax: confirm(“message”);
LESSON 2 CODES
If Statement
• It is used to check or verify a condition and execute a
set of statements only if the condition is true. This should
be referred to as a statement and not as a function.
Nested If
• "If statement" with another "If statement"
• Used when the "child" condition is only
checked when the parent condition is true.
If Else statement
• It has the same format as the "if statement,” the
only thing is that an Else part is added to an If statement.
So, if the condition satisfies the statement inside an If statement,
part of the code will be executed. And the Else statement inside
the Else part only will be executed.
If Else If statement
• It refers to an if statement that depends on another if statement.
When you use the If Else If statement, you are simply telling the
browser that something is True, then do something else, if the other
thing is True, then do something else, and so on.
