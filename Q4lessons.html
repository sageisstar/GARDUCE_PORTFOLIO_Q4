LESSON 7: FORMATTING LAYOUT
PROPERTY/VALUE:
1. position:static
2. position:relative
3. position:absolute
4. position:fixed 		DESCRIPTION:
1. default or normal position of an object if
one would not specify a position
2. position of the box relative to where the
default or normal position of an object
3. position is defined by the values where the
position of the element would be from the
specified reference point
4. same as the absolute position, the only
difference is that the object is “fixed” and
will not move with the scroll action of the page
